Abstract

This is a example of an ASPX file that displays a table populated with data from a MS SQL query via SqlDataAdapter. The SQL where cause is passed through the URL and displayed in a Textbox. It includes custom functions called MessageBox(mystring), LoadData(), and debug(mystring).
